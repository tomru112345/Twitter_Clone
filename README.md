## Railsアプリのカスタマイズ

### ビューおよびコントローラーの作成

```bash
rails g controller users index show
```

次のようなファイルが自動生成

```bash
app/controllers/users_controller.rb
app/views/users/index.html.erb
app/views/users/show.html.erb
```
